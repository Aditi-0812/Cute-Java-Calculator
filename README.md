# 🧮 Cute Calculator – Java Swing Project

A simple and aesthetic calculator made using **Java Swing**.  
Supports basic arithmetic operations with a clean GUI layout.

---

## ✨ Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 🖱️ square and 1/x 
- 🎨 Cute and minimal design (customizable)
